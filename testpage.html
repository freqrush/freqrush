Overview of Freqrush projects

freqrush: Sandbox on github where you find this page and these projects.

dir2tap: Perl script to add all samples from a folder to a TapeUTape preset.

dir2hydrogen: Perl script to add all samples from a folder to a Hydrogen Kit.

FreqrushFilterbank: SynthEdit files and GUI for Sherman Filterbank VST version. (NOT FINISHED)

RolandSPD11_JSynthLib: Roland SPD 11 Driver for JSynthLib
